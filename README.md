# BTC-Chat

A virtual assistant designed to help you understand everything about Bitcoin.

## Built with

frontend (this repository)

- React
- Vite
- Typescript
- Tailwind CSS

backend (private repository, at the moment)

- Node JS
- Express
- Typescript
- openAI API

## Features

[x] Q&A interfaces
[x] Support gpt-3.5-turbo
[-] Data stored locally
[-] Night/Dark mode
